# SMS-P2P-Poly-users

P2P-мессенджер на базе WebRTC для пользователей Poly, работающий по принципу групповой Mesh-сети.

## 🚀 Особенности проекта

- **Децентрализация (P2P):** Прямой обмен сообщениями между пользователями без промежуточного сервера базы данных.
- **Групповой Mesh-чат:** Устройства динамически соединяются друг с другом, формируя общую сеть покрытия.
- **Сквозное шифрование (E2E):** Полная конфиденциальность переписки — сообщения шифруются на стороне отправителя и расшифровываются только получателем.
- **Кастомные никнеймы:** Возможность установить уникальное отображаемое имя внутри сети.
- **WebSocket Сигналинг:** Быстрое первичное соединение пиров через WebSocket-сервер для установления WebRTC-сессии.

## 🛠 Технологический стек

- **Client (Android):** Java, Android SDK, WebRTC Android library, OkHttp (WebSocket Client)
- **Data Parsing:** JSON (`org.json.JSONObject`)
- **Backend (Signaling):** Node.js / WebSocket (`ws`)

## 🏗 Инструкция по сборке клиента (Android)

### Требования
- Android Studio Jellyfish (или новее)
- Android SDK (API level 26 или выше)
- Gradle 8.0+

### Пошаговая сборка:
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/canabis5w5/SMS-P2P-Poly-users.git
   cd SMS-P2P-Poly-users
   ```
2. Откройте проект в **Android Studio** (`File -> Open`).
3. Дождитесь окончания синхронизации Gradle.
4. Откройте файл `MainActivity.java` и укажите IP-адрес вашего запущенного сигнального сервера в переменной `wsServerUrl`:
   ```java
   private String wsServerUrl = "ws://<ВАШ_IP_АДРЕС>:8080";
   ```
5. Подключите Android-устройство по USB (или запустите эмулятор) и нажмите **Run ('app')** (сочетание `Shift + F10`).

## 🌐 Сигнальный WebSocket Сервер

Так как WebRTC требует обмена метаданными (SDP-офферами, ответами и ICE-кандидатами) перед установкой прямого соединения, используется легковесный сигнальный сервер на Node.js. Сервер работает как шина бродкаста, пересылая JSON-пакеты между участниками комнаты.

### Пример минимального сервера (`server.js`)

Вы можете развернуть этот код на хостинге или локально в вашей сети:

```javascript
const WebSocket = require('ws');

const wss = new WebSocket.Server({ port: 8080 }, () => {
    console.log("Signaling server started on port 8080");
});

wss.on('connection', (ws) => {
    ws.on('message', (message) => {
        // Пересылаем полученный пакет (SDP/ICE) всем остальным клиентам
        wss.clients.forEach((client) => {
            if (client !== ws && client.readyState === WebSocket.OPEN) {
                client.send(message.toString());
            }
        });
    });
});
```

### Запуск сервера:
1. Установите Node.js на компьютер/сервер.
2. Инициализируйте проект и установите библиотеку `ws`:
   ```bash
   npm init -y
   npm install ws
   ```
3. Сохраните код выше в файл `server.js` и запустите его:
   ```bash
   node server.js
   ```
